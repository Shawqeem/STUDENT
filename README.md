学生成绩管理系统：
登录
教务：录入课程信息、教师信息、学生信息，用链表进行数据存储。
教师端：录入成绩，修改成绩，统计成绩，查询成绩
学生端：查询成绩，打印成绩单等等


继承关系：

基类：

基类包含号码、得分、名字、密码除构造、析构、复制构造函数之外，还有获得信息函数；
派生类：教务，有新建课程/学生/教师的权力，需要获得新信息函数

需要预设教师/教务/的信息
登录->教务->选择新建课程/学生/教师->输入信息，固定格式->获得新信息->构造相应对象
         ->选择回到主界面->登录
         ->选择查看某课程/某教师/某学生的评分/成绩情况（包括rank）->打印评分表/成绩单
    ->教师->选择查看某学生的成绩情况（包括rank）/查看自己所教课程的评分情况->打印评分表/成绩单
         ->选择回到主界面->登录
    ->学生->选择查看自己的成绩情况（不包括rank）->->打印成绩单
          ->选择回到主界面->登录

导入时最开始要算一遍均分。

导入导出功能。
